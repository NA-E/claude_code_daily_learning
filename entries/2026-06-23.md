**"Slow Down to Speed Up": Meta Instagram Outage from AI-Generated + AI-Reviewed Code**
The Pragmatic Engineer's same report documented what happens when AI code generation outpaces human oversight: a Meta Instagram outage from AI-generated, AI-reviewed code allowed anyone to change any account's email — including Barack Obama's Instagram. Instagram's staffing context: 44% design headcount reduction; 95% headcount reduction in Developer Documentation and Support; 50% reduction in Trust and Safety. The outage is now cited as the clearest documented case of "AI code without human review going wrong at scale."
- **Source:** The Pragmatic Engineer (pragmaticengineer+deepdives@substack.com), June 23, 2026
- **Why it matters for Nourin:** The June 22 entry introduced DeepMind's Control Roadmap and the "prove your work before touching production" principle. The Meta Instagram case is the production incident that validates why that principle exists. It's not hypothetical risk — it happened, at scale, at a company that moved fast on AI code while cutting review staff.
- **Action:** If any of your Claude Code workflows touch authentication, permissions, email/account data, or financial logic: add a mandatory human review step for that specific code category. Use the DeepMind principle from June 22: "Show me what you'd do before doing it, then wait for my confirmation."
