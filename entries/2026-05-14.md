**Claude Skills — Reusable Instruction Sets, /skillname Trigger, 3-Part Build Structure**

The Neuron Daily (May 14) featured a tutorial from YouTube creator Alex Pereira: Claude Skills are permanent instruction sets installed once and triggered with a `/command` in any chat. Alex built 6 for his content workflow: `/hook-generator`, `/image-creator`, `/video-generator`, `/thumbnail-builder`, `/face-lock`, `/caption-writer`. Build one: Claude → Customize → + button → Create Skill. Structure: (1) description + when to trigger, (2) step-by-step instructions, (3) 2-3 examples of good output. **Why it matters for Nourin:** The distinction between a Skill and a saved prompt is precision: Skills auto-trigger based on context, prompts require you to remember and paste. Every recurring Claude task with a consistent output shape — including this digest format — is a Skill candidate.

**Action:** Identify your single most-used recurring Claude prompt this week. Run it through Pereira's 3-criteria test (recurring? structured? consistent output format?).
