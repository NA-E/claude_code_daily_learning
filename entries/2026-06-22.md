**Perplexity Brain: Overnight Memory That Builds a Context Graph and Lifts Accuracy by 25%**
Perplexity launched Brain, an agent memory system that builds a context graph of its prior work and continues learning overnight. Stated accuracy improvement: 25% versus the same model without Brain. Unlike static memory stores, Brain specifically models how the agent's tasks connect to each other — not just what facts were stated, but what work was done and why. It learns between sessions without requiring the user to define what to remember.
- **Source:** The AI Break (theaibreak@substack.com) — "ChatGPT Enterprise Just Got Spend Controls," June 22, 2026; The Neuron Daily (June 22, 2026)
- **Why it matters for Nourin:** The June 20 entry introduced the context-management ladder (convention drift vs. context overflow). Perplexity Brain is a production implementation of autonomous convention-drift solving — the model learns your patterns without you writing CLAUDE.md rules. Worth watching how the Claude ecosystem responds to this pattern.
- **Action:** If you use Perplexity for research or summarization tasks, enable Brain in Perplexity settings and run it for one week. At the end of the week, test whether it recalls your project preferences and domain context without re-prompting.
