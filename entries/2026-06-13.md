### **WORKFLOW**

**Ambient Agent Architecture — Kafka + Ollama Two-Stage Funnel, Coinbase WebSocket, Notify-Only (The Neural Maze)**
The Neural Maze Issue #05 published the clearest end-to-end architecture for a locally-runnable ambient agent. Stack: **Coinbase public WebSocket** (live crypto trades, no API key required) → **Kafka** (message bus, decouples senses from brain) → **local Ollama LLM** (no cloud, no bill) → **Telegram or terminal notification**. The core design insight: a **two-stage funnel** to control cost: Stage 1 — cheap math runs on every time window (% price move, trade count, buy/sell imbalance, volume) with no LLM; if nothing crosses a threshold, the window is discarded for free. Stage 2 — LLM called **only when Stage 1 trips** (a sharp move or lopsided order flow). The mental model: "The LLM is precious; spend it deliberately." Escalation level: **Notify only** (agent watches, reasons, informs — never takes irreversible action). The four-step loop: Observe → Infer → Decide → Notify.
*Source: The Neural Maze (theneuralmaze@substack.com), Issue #05 "Building a Local Ambient Agent That Never Sleeps," June 13, 2026 (theneuralmaze.substack.com/p/building-a-local-ambient-agent-that)*
**Why it matters for you:** This two-stage funnel architecture is the answer to the token cost problem that has appeared in every entry since 2026-05-16: spend frontier model tokens only on events that pass a cheap pre-filter. The specific stack (WebSocket → Kafka → Ollama) is reproducible on a laptop. More importantly, the same architecture applies to anything where you want a model watching a stream: emails, PRs, log files, Slack messages. Repoint the producer (the WebSocket) and you've built a different ambient agent.
**Action:** Follow the Neural Maze walkthrough at theneuralmaze.substack.com/p/building-a-local-ambient-agent-that. Build the Coinbase → Kafka → Ollama stack as described. Then replace the Coinbase WebSocket with a source that matters for your workflow (email inbox, GitHub events, Slack channel). The two-stage funnel pattern is transferable directly.
