**Run the "Grill Me" context interview before starting your next Claude Code project.**

Here is the exact prompt to start it manually if you don't have the skill yet — paste this into a new Claude session:

```
You are going to interview me to build a reusable context file.
Ask me one question at a time. Each question should help you
understand how I think, decide, and communicate.
After each answer, acknowledge it, then ask the next question.
Keep going until you can describe my voice, my decision-making
framework, my preferred communication style, and my professional
context with specific examples. Do not stop until you have
collected at least 10 clear data points.
Then write the results to a markdown file called context/me.md.
```

Run this interview (15–30 minutes). Save the output to `context/me.md` in your current project. Load it at the start of every subsequent Claude Code session: `cat context/me.md | claude --system-prompt -`.

- The reusable `context/me.md` is a one-time 30-minute investment that improves every Claude Code session you run from this point forward.
