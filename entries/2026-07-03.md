**The "Planner and Judge, Not Executor" Fable 5 Prompt Template**
*Source: The Neuron Daily ("AI Skill of the Day: How to Prompt Fable 5", theneuron@newsletter.theneurondaily.com)*

A second, independently-sourced prompt template for Fable 5 (Anthropic's own prompting docs, translated by The Neuron): structure any Fable 5 call as "planner and judge, not executor" — give it the goal and context, then have it (1) write a concise plan with exact files/steps/risks/success criteria, (2) write a handoff brief for a cheaper executor model, (3) list what Fable should NOT do itself, (4) define a memory note to save, (5) define a verification gate for the executor to pass, (6) review the executor's finished result as judge and list only ship-blocking issues.

*Why it matters for you*: This is a concrete, copy-pasteable structure that operationalizes yesterday's McFarland rules (act, verify, separate looking from doing) into an actual reusable prompt rather than abstract guidance.

*Action*: Save this exact 6-step template as a Claude Code skill or snippet, and use it the next time you reach for Fable 5 instead of writing an ad hoc prompt.
