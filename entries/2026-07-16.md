**OpenAI's Scheduled-Task vs. Workspace-Agent Decision Rule**
*Source: The Neuron, "AI Skill of the Day"*

OpenAI's own guidance: for any recurring task, first ask whether it mainly needs connected tools (Slack, email, calendar, Drive, Notion, Docs) — if so, make it a scheduled task. *Why it matters*: directly describes the shape of this very digest routine — it depends on the journal MCP skill and a git-commit workflow, not just connected tools, which by OpenAI's own rule means it's correctly built as an agent/routine rather than a simple scheduled task.

*Action*: run this exact classification question against any recurring task you're doing manually or considering automating, before building it the wrong way.
