**`/handoff` SKILL.md — Context Preservation Across Claude Sessions**
Matt Pocock open-sourced a `/handoff` skill at `github.com/mattpocock/skills` (confirmed as The Neuron Daily's skill-of-the-day feature, May 10). When you're near Claude's context limit mid-task, type `/handoff` and the skill compacts the session into a structured handoff doc: current goal, sub-goals, established context, artifacts produced (with paths), decisions made and why, what's blocking, and suggested next steps. Works for coding, research, writing, strategy — any task where value compounds across turns.
```
/handoff
Compact this session into a clean handoff document. Include:
- Current goal and sub-goals
- Context that took us multiple turns to establish
- Artifacts produced so far (with links/paths)
- Decisions made and why
- What's blocking, if anything
- Suggested next steps for whoever picks this up
```
**Source:** The Neuron Daily — `theneurondaily.com`, skill-of-the-day, May 10. Repo commit confirmed: `github.com/mattpocock/skills/blob/733d312884b3878a9a9cff693c5886943753a741/skills/in-progress/handoff/SKILL.md`.
**Why it matters for you:** Every long Claude Code session hits context limits. This is the fix — free and installable in 60 seconds.
**Action:** Go to `github.com/mattpocock/skills`, grab `SKILL.md`, add it to your Claude project (Settings → Skills → Add), then type `/handoff` at the end of any long session.
